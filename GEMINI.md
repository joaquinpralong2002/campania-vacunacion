# Resumen del Directorio

Este directorio contiene el proyecto para una simulación de campaña de vacunación, como se describe en el documento "Calculo Numerico - Taller 2025.pdf". El objetivo es crear un modelo de simulación de eventos discretos para analizar el proceso de vacunación, considerando factores como el número de puestos de vacunación, las tasas de llegada y los costos.

# Archivos Clave

*   **Calculo Numerico - Taller 2025.pdf**: Este documento describe los requisitos para el proyecto de simulación de la campaña de vacunación. Incluye el contexto, las consideraciones, las tareas, los entregables y los criterios de evaluación.
*   **diagrama-flujo-vacunacion.jpeg**: Diagrama de flujo que representa el proceso de vacunación.
*   **Trabajo Final Integrador.md**: Este archivo contiene las consignas del proyecto de forma más clara y detallada.

# Tecnologías Seleccionadas

*   **Lenguaje**: Python
*   **Librerías**:
    *   **SimPy**: Para la simulación de eventos discretos.
    *   **Pandas** y **NumPy**: Para el análisis y manipulación de datos.
    *   **Matplotlib** / **Seaborn**: Para la visualización de datos.

# Uso

El propósito principal de este directorio es desarrollar la simulación de la campaña de vacunación. Esto implica:

1.  **Modelado**: Diseñar e implementar un modelo de simulación de eventos discretos basado en las especificaciones del documento PDF.
2.  **Experimentación**: Ejecutar diferentes escenarios de simulación para analizar el rendimiento de la campaña de vacunación en diversas condiciones.
3.  **Análisis**: Analizar los resultados de la simulación para sacar conclusiones y hacer recomendaciones.
4.  **Reporte**: Crear un informe final con los hallazgos, incluyendo gráficos y tablas, así como el código fuente de la simulación.

El flujo de trabajo sugerido es: 1. Diseñar el modelo. 2. Implementar la simulación con Python y SimPy. 3. Ejecutar experimentos con diferentes escenarios. 4. Analizar y visualizar los resultados.
